BeautifyJson

A jQuery plug-in that takes a JSON string and makes it colorful and easy to read.

Example:

BeautifyJson turns this

{"object1" : {"innerObject" : {"string1" : "test","string2" : "moreTest","nothing" : null},"niceLinks" : ["http://google.com","http://en.wikipedia.org"],"thisIsTrue" : false},"anArray" : [42,23,15]}

into this

{"object1" : {"innerObject" : {"string1" : "test","string2" : "moreTest","nothing" : null},"niceLinks" : ["http://google.com","http://en.wikipedia.org"],"thisIsTrue" : false},"anArray" : [42,23,15]}

Configuration

Call $(element).BeautifyJson(); with a selector that points to a div (or any block element) containing only correctly formatted JSON.

After calling BeautifyJson, the plaintext is available in $(element).data('bjson-plain').

Options

hover: boolean. Default: false. Enables hover-effects, like in the example below.
links: boolean. Default: true. Whether a string that is a URL should be clickable.
indent: integer. Default: 30. The number of pixel for each indentation.
wrapper: string. Default: "%bjson%". If the JSON should be wrapped in a function call. %bjson% is the placeholder for the BeautifyJson output. See example below.

Weird self-referencing example:

{ "hover" : true, "links" : false, "indent" : 40, "wrapper" : "$('#ex2').BeautifyJson(%bjson%);" }
